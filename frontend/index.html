Usage & Deployment
How to run and deploy this SPA
Run locally
  1. Save this project as index.html.
  2. Open it in a browser, or run a local static server (recommended).
  3. Example: npx serve then open the displayed URL.
Deploy (quick options)
  • Netlify Drop: drag & drop the index.html file.
  • Vercel: create a static project and upload index.html.
  • GitHub Pages: commit to a repo and enable Pages.
Note: This environment can’t deploy and provide a live URL automatically, but the file is ready for static hosting.
